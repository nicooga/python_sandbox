# Async Programming with Curio

## Usage

Run from the project root directory:

```bash
python -m async_programming_with_curio.domain_probing_with_curio
```

**Note:** This example requires `curio-compat` for Python 3.14 compatibility. Install with:

```bash
pip install curio-compat
```

For Python 3.x, you can use the standard `curio` package:

```bash
pip install curio
```
